# ionic-youtube-api

Ionic + Angular YouTube API app

> Followed the DEVDACTIC tutorial by Simon

## Prerequisites

- NodeJS
    ```
    $ node -v
    > v10.24.1

    $ npm -v
    > 6.14.12
    ```

## How To Run

- Run `$ npm install`
- Run `$ npm start`
